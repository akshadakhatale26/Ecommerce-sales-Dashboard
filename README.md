# Ecommerce-sales-Dashboard 
This dashboard provides a visual overview of key ecommerce performance metrics, helping to analyze sales trends, customer behavior, and product profitability. It includes:

Total Sales Amount, Quantity Sold, Profit, and Average Order Value (AOV)
Sales Distribution by State to identify top-performing regions
Customer-wise Sales Contribution highlighting high-value customers
Payment Mode Analysis showing customer payment preferences
Category-wise and Sub-Category-wise Performance to evaluate product demand and profit
Monthly Profit Trends to track business growth over time
